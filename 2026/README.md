# 2026

Event-specific planning and the meal plan for 2026.

This is a `<year>/` directory: it holds everything to do with this year's
event. There's no fixed schema — grow whatever sub-structure the year needs.
A common shape:

- `menu/` — the meal-plan spine and per-day detail
- a per-day file each for the days the kitchen runs
- a quantities / shopping file for aggregating the shop
- whatever else helps (equipment, prep timeline, headcounts)

Dishes link out to the reusable cards in [`../recipes/`](../recipes/) rather
than restating the recipe here.

## Planning files

- [`equipment.md`](equipment.md) — confirmed kitchen kit: what's on site, and
  what members bring.
